# Тест для GreenApi

**Как запустить проект**

1. Клонировать репозиторий

   ```bash
   git clone https://github.com/komubosu/test-for-green-api.git
   ```

2. Установить зависимости

   ```bash
   yarn
   ```

3. Запустить виртуальный сервер

   ```bash
   yarn dev
   ```

**Что используется?**

- TypeScript
- React
- scss
- [GreenApi](https://green-api.com/)
